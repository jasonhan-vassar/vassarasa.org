urop.guide

Website / Github

I created urop.guide with a friend (Gilbert Yan) to help undergraduates at MIT find research opportunities.

The MIT published urop listings are great, but are missing some crucial features like bookmarking urops, searching for keywords, and filtering by term. We created urop.guide to address these problems.

Our site has two components; the web scraper where we obtain information on the urops from MIT's advertised listings and the website where we display this information. The web scraper is run every night at midnight so our listings remain up to date. I primarily worked on the web scraper, UI design, and publicity for the website.

If you have any questions or suggestions for urop.guide, please let us know!
