MIT Print Bot

Link

I created MIT Print Bot with some friends (Gilbert Yan, Jing Lin, and Tony Wang) to help students at MIT print to MIT printers from any device, anywhere.

The MIT Athena printers require you to be connected to one of MIT's networks in order to print to them. This makes it inconvenient if you live in an apartment off campus. Additionally, in order to print you need to install a specialized driver, making it very difficult to print from a mobile device.

We created MIT Print Bot to solve these problems. MIT Print Bot is a Facebook messenger bot connected to the Athena printers. It prompts users to enter the information needed to print, then sends the file to the printer.

If you have any questions or suggestions for MIT Print bot, please let us know!
